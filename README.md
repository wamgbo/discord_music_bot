# Discord Music Bot

A Discord music bot supporting **YouTube streaming** and **Bilibili download & play**, with interactive control buttons for pause/resume, skip, loop, and stop.

---

## Features

- 🎵 Play music from **YouTube** (direct stream, no download)  
- 🎶 Play Bilibili audio (download & convert with yt-dlp)  
- ⏯️ Interactive control buttons (pause/resume, loop, skip, stop)  
- 📜 Queue management  
- 🔁 Single-track loop support  
- 💬 Slash commands (`/play`, `/list`)  

---

## Requirements

- Python **3.10+**  
- FFmpeg installed & in PATH  
- Discord bot token  
- Dependencies listed in `requirements.txt`
